# Activity Data Model

Data structures for tracking user activity, engagement feeds, and platform interactions.

## Activity Types

<AccordionGroup>
<Accordion title="Trading Activity">
  - Trade executions (buy/sell)
  - Order placements and cancellations
  - Liquidity provision events
  - Price alerts triggered
</Accordion>

<Accordion title="Social Activity">
  - Comments posted
  - Votes cast (upvote/downvote)
  - Token favorites added/removed
  - User follows and connections
</Accordion>

<Accordion title="Token Activity">
  - Token creation
  - Token updates (image, description)
  - Achievement unlocks
  - Referral events
</Accordion>
</AccordionGroup>

</ResponseExample>
