# Token Comments API

Manage community discussions and social features around tokens with comments, upvotes, and moderation tools.

## Get Token Comments

<ParamField path="GET /token/{id}/comments" type="endpoint">
Retrieve comments for a specific token with pagination and filtering.
</ParamField>

<ParamField path="id" type="string" required>
Token identifier.
</ParamField>

<ParamField query="page" type="number" default="1">
Page number for pagination.
</ParamField>

<ParamField query="limit" type="number" default="10">
Number of comments per page (1-100).
</ParamField>

<ParamField query="user" type="string">
Filter comments by specific user.
</ParamField>

<ParamField query="pinned" type="boolean">
Filter for pinned comments only.
</ParamField>

<ParamField query="community" type="integer" default="0">
Filter for community comments (0 for regular, 1 for community).
</ParamField>

## Post Token Comment

<ParamField path="POST /token/{id}/comment" type="endpoint">
Create a new comment on a token.
</ParamField>

<ParamField body="message" type="string" required>
The comment content.
</ParamField>

<ParamField body="community" type="integer" default="0">
Whether this is a community comment (requires token ownership).
</ParamField>

<RequestExample>
```bash cURL
curl -X POST 'https://api.odin.com/token/btc/comment' \
  -H 'Authorization: Bearer YOUR_JWT_TOKEN' \
  -H 'Content-Type: application/json' \
  -d '{
    "message": "Great project with strong fundamentals!",
    "community": 0
  }'
```
</RequestExample>

## Comment Moderation

### Pin Comment

<ParamField path="POST /token/{token}/comment/{comment}/pin" type="endpoint">
Pin a comment to the top of the discussion (admin only).
</ParamField>

### Unpin Comment

<ParamField path="POST /token/{token}/comment/{comment}/unpin" type="endpoint">
Remove pin status from a comment (admin only).
</ParamField>

## Comment Voting

<ParamField path="POST /token/{tokenId}/comment/{id}/{action}" type="endpoint">
Vote on a comment (upvote/downvote).
</ParamField>

<ParamField path="action" type="string" required>
Vote action: "upvote" or "downvote".
</ParamField>

<ParamField path="DELETE /token/{tokenId}/comment/{id}/upvote" type="endpoint">
Remove your vote from a comment.
</ParamField>
