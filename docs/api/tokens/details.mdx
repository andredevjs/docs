# Token Details API

Access detailed information about specific tokens including comprehensive metadata, market analytics, and real-time metrics.

## Get Token by ID

<ParamField path="GET /token/{id}" type="endpoint">
Retrieve complete information about a specific token.
</ParamField>

<ParamField path="id" type="string" required>
The unique token identifier.
</ParamField>

<RequestExample>
```bash cURL
curl -X GET 'https://api.odin.com/token/btc' \
  -H 'Authorization: Bearer YOUR_JWT_TOKEN'
```
</RequestExample>

<ResponseExample>
```json Success
{
  "data": {
    "id": "btc",
    "name": "Bitcoin Token",
    "description": "A Bitcoin-backed token on Odin",
    "price": 50000,
    "marketcap": 1000000000,
    "volume": 5000000,
    "bonded": false,
    "featured": true,
    "verified": true,
    "holder_count": 1250,
    "twitter": "@bitcointoken",
    "website": "https://bitcointoken.com"
  }
}
```
</ResponseExample>

## Get Token Image

<ParamField path="GET /token/{id}/image" type="endpoint">
Retrieve the token's image/logo as a binary stream.
</ParamField>

<RequestExample>
```bash cURL
curl -X GET 'https://api.odin.com/token/btc/image' \
  --output btc-logo.png
```
</RequestExample>

## Token Verification

<ParamField path="GET /token/{id}/twitter/verify" type="endpoint">
Verify a token's Twitter account integration.
</ParamField>

<ParamField query="code" type="string" required>
Twitter verification code.
</ParamField>

<ParamField query="redirect_uri" type="string" required>
Redirect URI for OAuth flow.
</ParamField>
