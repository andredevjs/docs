# User Balances & Portfolio

Monitor user token holdings, portfolio values, and asset distributions across the platform.

## Get User Statistics

<ParamField path="GET /user/{principal}/stats" type="endpoint">
Get comprehensive user statistics including portfolio value and trading metrics.
</ParamField>

<ResponseExample>
```json User Portfolio Stats
{
  "data": {
    "total_asset_value": 150000000,
    "total_liquidity": 25000000,
    "btc": 75000000,
    "portfolio_change_24h": 5.2,
    "trading_volume_30d": 500000000,
    "profit_loss_total": 25000000,
    "active_positions": 8,
    "created_tokens": 3
  }
}
```
</ResponseExample>

## Get User Token Holdings

<ParamField path="GET /user/{principal}/tokens" type="endpoint">
Get all tokens owned by a user with current balances and values.
</ParamField>

## Get User's Liquidity Positions

<ParamField path="GET /user/{principal}/liquidity" type="endpoint">
Get user's liquidity provider positions across all pools.
</ParamField>

## Get Created Tokens

<ParamField path="GET /user/{principal}/created" type="endpoint">
Get tokens created by the user.
</ParamField>

## Get Favorite Tokens

<ParamField path="GET /user/{principal}/token/favorites" type="endpoint">
Get user's favorite tokens list.
</ParamField>
