# User Profile Management

Create and manage user profiles including usernames, bios, profile images, and social integrations.

## Get User Profile

<ParamField path="GET /user/{principal}" type="endpoint">
Retrieve complete user profile information.
</ParamField>

## Update Username

<ParamField path="POST /user/username" type="endpoint">
Update the authenticated user's username.
</ParamField>

<ParamField body="username" type="string" required>
New username (must be unique and follow platform guidelines).
</ParamField>

<RequestExample>
```bash cURL
curl -X POST 'https://api.odin.com/user/username' \
  -H 'Authorization: Bearer YOUR_JWT_TOKEN' \
  -H 'Content-Type: application/json' \
  -d '{"username": "new_username"}'
```
</RequestExample>

## Update Profile Bio

<ParamField path="POST /user/bio" type="endpoint">
Update the authenticated user's profile bio.
</ParamField>

<ParamField body="bio" type="string" required>
Profile bio (maximum 500 characters).
</ParamField>

## Profile Image Management

### Upload Profile Image

<ParamField path="POST /user/image" type="endpoint">
Upload a new profile image.
</ParamField>

<ParamField body="image" type="file" required>
Image file (JPEG, PNG, WebP, max 5MB).
</ParamField>

### Get Profile Image

<ParamField path="GET /user/{principal}/image" type="endpoint">
Retrieve user's profile image.
</ParamField>
