# User Achievement System

Engage users with a comprehensive achievement system featuring progress tracking, rewards, and gamification elements.

## Get User Achievements

<ParamField path="GET /user/{principal}/achievements" type="endpoint">
Get user's achieved and available achievements with progress tracking.
</ParamField>

        "points": 100,
        "achieved_at": "2024-01-15T10:30:00Z"
      }
    ],
    "available": [
      {
        "id": "volume_trader",
        "name": "Volume Trader",
        "description": "Trade $10,000 in volume",

  }
}

</ResponseExample>
